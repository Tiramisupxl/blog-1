#### 问题描述
想问个特简单的问题（如下代码优化下大量的if - else）：太繁琐
 let x, y, z, m
      if (x > 0) {
        if (y === 1) {
          m = 1
          return 2
        } else {
          if (z === 3) {

#### 问题描述
大家是怎么理解 peerDependency  这个东西的


#### 问题解决
可以类比一下，例如 koa或者 express 的middleware脱离koa或express不能独自使用，但是如果项目已经安装koa或express，则可以正常使用。那么这些middleware库的peerDependency就是koa或者express

rollup-plugin-xxx也是一样的 这些库不需要用到rollup的方法，但是却不能脱离rollup运行，所以他们的peerDependency就是rollup
